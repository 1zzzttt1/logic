# Claude Code 入门指南

## 什么是 Claude Code？

Claude Code 是 Anthropic 开发的 AI 编程助手，它能够理解代码上下文，帮助开发者编写、审查和调试代码。

## 安装与配置

### 环境要求

- Node.js 18+
- npm 或 yarn
- 代码编辑器（推荐 VS Code）

### 安装步骤

1. 全局安装 Claude Code CLI
2. 配置 API 密钥
3. 初始化项目

## 基本使用

### 交互模式

Claude Code 支持交互式对话，你可以：
- 询问代码相关问题
- 请求代码审查
- 生成代码片段

### 自动化任务

Claude Code 还可以用于自动化常见开发任务，如：
- 重构代码
- 生成文档
- 编写测试

## 最佳实践

> 关键在于把 AI 当作一个经验丰富的同事，而不是一个黑盒子。

学会正确地与 AI 协作将极大提升你的开发效率。
