## 项目定位

这是一个 **个人 AI 知识博客网站**。

### 产品定位

一个记录 AI 学习笔记、分享技术见解的个人知识博客网站。

### 核心理念

知识沉淀 + 公开分享，双轨并行。

### 它是什么

- 静态内容网站
- 个人知识管理工具
- 面向 AI 学习者 / Claude Code / Cursor / AI 应用开发方向开发者的技术博客

## 知识库页面布局（已经确定）

知识库采用 **三栏式文档布局**：

- 左侧：一级分类导航，分类可展开/收起，展开后显示该分类下的文章列表
- 中间：当前选中文章的正文详情
- 右侧：当前文章目录 TOC

### 知识库交互方式

- 左侧分类可以展开/收起
- 展开后可以看到该分类下文章
- 点击文章后，中间区域切换为该文章正文
- 右侧目录基于当前文章内容生成
- 点击目录项可跳转到正文对应位置

### 默认状态

- 第一个分类默认展开
- 第一篇文章默认选中
- 中间默认展示第一篇文章正文
- 右侧默认展示第一篇文章目录

### 移动端方向

目前大方向是：

- 正文优先
- 分类与目录在移动端折叠
- 后续更推荐做成顶部两个按钮：分类 / 目录，各自打开抽屉
