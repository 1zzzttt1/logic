# 个人 AI 知识博客 - 页面蓝图设计

*基于已确认的产品规格文档（最终版）*

---

## 1. 站点地图 Sitemap

```
/
├── 首页 (Home)
│   └── 核心入口：学习 AI CTA
├── 知识库 (Knowledge)
│   └── [三栏文档布局]
│       ├── 左侧：分类导航 + 文章列表
│       ├── 中间：正文内容
│       └── 右侧：目录 TOC
│
├── 文章分享 (Articles)
│   ├── /articles (列表页)
│   └── /articles/:id (详情页)
│
└── 关于我 (About)
```

**页面跳转关系**：

```
首页
└── → 知识库（唯一 CTA：学习 AI）

知识库
├── → 文章分享（列表页）
├── → 关于我
└── 内部：切换分类 / 切换文章

文章分享列表页
├── → 知识库
├── → 关于我
└── → 文章详情页

文章详情页
├── → 知识库
├── → 关于我
└── → 原文链接（外部）

关于我
├── → 知识库
└── → 文章分享（列表页）
```

## 2. 页面蓝图总览

### 2.1 首页

| 项目 | 内容 |
|------|------|
| **页面目标** | 传达站点定位，聚焦引导用户开始学习 AI |
| **路由** | `/` |
| **区块结构** | Header → Hero → CTA |

**区块详情**：

| 区块 | 位置 | 作用 | 主要内容 |
|------|------|------|----------|
| Header | 顶部固定 | 全局导航 | Logo + 导航链接（知识库 / 文章分享 / 关于我） |
| Hero | 主体顶部 | 传达站点定位和学习路径感 | 站点名称 + Slogan + 一句话描述 |
| 学习 AI CTA | Hero 下方 | 唯一核心行动引导 | 大按钮：开始学习 → 跳转知识库 |

**用户操作**：
1. 点击导航链接跳转
2. 点击 CTA 按钮跳转到知识库

**区块详情**：

| 区块 | 位置 | 作用 |

**详细结构见第 3 节**

## 3. 知识库页面重点细化

### 3.1 整体布局

```
+----------------------------------------------------------+
|  Header (固定)                                           |
+----------------------------------------------------------+
|          |                                  |              |
|  左侧    |         中间                     |   右侧      |
|  20%    |         55%                      |   25%       |
|          |                                  |              |
| 分类导航 |         正文                      |   目录      |
| +文章树  |                                  |   TOC       |
|          |                                  |              |
+----------------------------------------------------------+
```

### 3.2 左侧：分类导航 + 文章树

**结构**：

```
左侧边栏 (20% 宽度)
├── 分类 1 [AI基础] ▼
│   ├── 文章 1.1: 什么是 token
│   ├── 文章 1.2: 什么是 prompt engineering
│   └── 文章 1.3: ...
├── 分类 2 [Claude Code 使用教程] ▲
│   ├── 文章 2.1: 入门指南
│   └── 文章 2.2: 常用命令
└── 分类 3 [AI应用开发] ▲
    └── ...
```

**交互状态**：

| 状态 | 表现 |
|------|------|
| 分类收起 | 显示分类名称 + 展开箭头 ▼ |
| 分类展开 | 显示分类名称 + 收起箭头 ▲ + 文章列表 |
| 文章未选中 | 正常文字颜色 |
| 文章选中 | 高亮背景 + 粗体 |
| 空分类 | 显示"暂无文章" |

### 3.3 中间：正文区域

**结构**：

```
中间正文区域 (55% 宽度)
├── 当前位置 (面包屑)
│   └── 知识库 > AI基础 > 什么是 token
├── 文章标题
│   └── 什么是 token
├── 正文内容
│   └── Markdown 渲染后的内容
└── [返回顶部按钮] (可选)
```

**内容规范**：
- 标题：H1
- 正文：标准 Markdown 渲染
- 代码块：等宽字体显示
- 链接：可点击跳转

### 3.4 右侧：目录 TOC

**结构**：

```
右侧目录 (25% 宽度)
├── 当前文章标题
├── 目录
│   ├── 什么是 token？
│   ├── 为什么重要？
│   ├── 如何计算？
│   └── 实际应用
└── [滚动跟随]
```

**交互状态**：
- 基于文章 H1/H2/H3 自动生成
- 点击目录项：平滑滚动到正文对应位置
- 阅读位置高亮：当前阅读的章节对应的目录项高亮

### 3.5 默认进入时的状态

| 项目 | 默认状态 |
|------|----------|
| 第一个分类 | 自动展开 |
| 第一篇文章 | 自动选中并显示在中间 |
| 右侧目录 | 显示第一篇文章的目录 |
| 页面滚动位置 | 置顶 |

### 3.6 切换行为

| 操作 | 行为 |
|------|------|
| 点击分类展开/收起 | 展开/收起该分类，显示/隐藏文章列表 |
| 点击分类下的文章 | 中间正文切换为该文章内容，右侧目录更新 |
| 点击目录项 | 平滑滚动到正文对应位置，同时更新目录高亮 |
| 切换分类 | 展开新分类，收起旧分类，选中新分类的第一篇文章 |

### 3.7 移动端适配

| 断点 | 布局调整 |
|------|----------|
| PC (≥1024px) | 三栏并排 |
| 平板 (768-1023px) | 两栏：左侧隐藏→可切换，正文+目录 |
| 手机 (<768px) | 单栏：顶部 tabs 切换分类/目录，正文区域全屏 |

**移动端具体方案**：
- 左侧边栏：变为底部抽屉或顶部 Tab 切换
- 右侧目录：默认收起，点击按钮展开/浮动
- 正文区域：全屏占据主要空间

## 4. 首页重点细化

### 4.1 Hero 区

**承载信息**：
1. 站点名称：个人 AI 知识博客
2. Slogan：传达学习路径感（如"从入门到实战，系统学习 AI"）
3. 一句话描述：强调知识沉淀（如"记录学习心得，整理知识体系，分享技术见解"）

**布局**：居中大标题 + 副标题 + CTA 按钮

### 4.2 学习 AI CTA

**唯一核心行动引导**：

| 按钮 | 描述 | 跳转目标 |
|------|------|----------|
| 开始学习 | 点击进入知识库，开始 AI 学习之旅 | /knowledge |

**按钮样式**：
- 大尺寸、醒目
- 文案："开始学习" 或 "学习 AI"

### 4.3 首页定位

| 区块 | 状态 |
|------|------|
| 精选推荐区 | ❌ 已删除 |
| 导航入口卡片 | ❌ 已删除 |
| 导航入口（Header） | ✅ 保留，用于页面间跳转 |

## 5. 可复用组件识别

### 5.1 通用组件（全局可用）

| 组件名 | 描述 | 适用页面 |
|--------|------|----------|
| **Header** | 顶部导航栏，包含 Logo + 导航链接 | 所有页面 |
| **NavLink** | 导航链接组件（支持激活状态） | Header 内部 |
| **Markdown 正文容器** | Markdown 渲染容器 | 知识库正文、文章详情 |

### 5.2 列表类组件

| 组件名 | 描述 | 适用页面 |
|--------|------|----------|
| **ArticleCard** | 文章卡片（标题 + 摘要 + 元信息） | 文章列表页 |
| **ArticleList** | 文章列表容器 | 文章列表页 |

### 5.3 知识库专用组件

| 组件名 | 描述 | 适用页面 |
|--------|------|----------|
| **CategoryNav** | 左侧分类导航容器 | 知识库页面 |
| **CategoryItem** | 单个分类项（支持展开/收起） | CategoryNav 内部 |
| **ArticleTree** | 分类下的文章列表 | CategoryItem 内部 |
| **ArticleTreeItem** | 文章列表项（支持选中状态） | ArticleTree 内部 |
| **TOC** | 目录组件（右侧） | 知识库页面 |
| **TOCItem** | 目录项（支持高亮状态） | TOC 内部 |
| **Breadcrumb** | 面包屑导航 | 知识库中间区域 |

### 5.4 文章详情专用组件

| 组件名 | 描述 | 适用页面 |
|--------|------|----------|
| **ArticleHeader** | 文章头部（元信息） | 文章详情页 |
| **ArticleContent** | 文章正文容器 | 文章详情页 |
| **SourceLink** | 原文链接按钮 | 文章详情页 |
| **TagList** | 标签列表 | 文章详情页 |

### 5.5 关于我专用组件

| 组件名 | 描述 | 适用页面 |
|--------|------|----------|
| **AuthorProfile** | 作者信息卡片（头像 + 简介） | 关于我页面 |
| **ContactList** | 联系方式列表 | 关于我页面 |
| **ContactItem** | 单个联系方式 | ContactList 内部 |

## 附录：页面蓝图交付说明
